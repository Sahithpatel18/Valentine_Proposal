## Run Locally

**Prerequisites:**  Node.js


1. Install dependencies:
   `npm install`
2. Set the `GEMINI_API_KEY` in [.env.local](.env.local) to your Gemini API key
3. Run the app:
   `npm run dev`

# Valentine_Proposal
Valentine Proposal Magic is an interactive AI-powered web app that creates a personalized digital proposal experience. Users can add names, photos, animations, and AI-generated love letters. When the recipient clicks “YES,” the sender receives an instant Telegram notification, making the moment memorable.
